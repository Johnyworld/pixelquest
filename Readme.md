# Pixel Quest (가제)

socket.io와 canvas를 이용한 간단한(?) 게임을 제작.

###

주요 기술

- HTML canvas
- socket.io
- typescript

# How to use

```jsx
// start socket server
npm run dev
or 
yarn dev
```

```jsx
// build typescript to js
npm run build
or 
yarn build
```

```jsx
// start development server
npm run start
or 
yarn start
```
